ttlcd-cpp

Modern C++ port of the original Python driver for the Thermaltake Tower 200 (and compatible models) 3.9-inch USB LCD panel (480×128 resolution). A highly customizable, performant system monitoring dashboard with live updates, custom TTF fonts, bar graphs, network speed, and more.

Features

Live CPU & RAM usage bars with network speed
Dynamic text widget with powerful placeholders
Custom TTF fonts via OpenCV FreeType (Digital-7, Orbitron)
Graceful error handling — shows ERR on screen on failure
Full JSON configuration for layout, color & widgets
Complete USB protocol with initialization sequence
Robust libusb error reporting
Temporary image rendering with automatic cleanup

Requirements

Runtime Dependencies

sudo apt install libusb-1.0-0 libopencv-core4.5 libopencv-imgproc4.5 \
    libopencv-imgcodecs4.5 libopencv-highgui4.5

Build Dependencies

sudo apt install build-essential pkg-config libusb-1.0-0-dev \
    libopencv-dev libfreetype6-dev

Building

make -j$(nproc)           # Build the executable
sudo ./ttlcd config.json  # Run (requires sudo for USB access)

Clean rebuild:

make clean && make -j$(nproc)

Install

wget https://github.com/YOURUSERNAME/ttlcd/releases/latest/download/ttlcd-v1.0.0-amd64.deb
sudo dpkg -i ttlcd-v1.0.0-amd64.deb
sudo rpm -i ttlcd-1.0.0-1.x86_64.rpm
# Download PKGBUILD.tar.gz from releases, extract then:
makepkg -si

Release

Tag a version to trigger an automated build and release for all 3 package managers via GitHub Actions.

git tag v1.0.0
git push origin v1.0.0

GitHub Actions will automatically build and publish .deb, .rpm, and PKGBUILD as release assets.

Configuration

Full config.json example:

{
  "idVendor": 9802,
  "idProduct": 9021,
  "use": "NODE",
  "orientation": "TOP",
  "background": "./bg.jpg",

  // Global defaults
  "font_size": 14,
  "font_color": [255, 255, 255],
  "font_file": "fonts/Digital-7.ttf",

  // Widgets
  "enable_date": true,
  "date_x": 20, "date_y": 10, "date_font_size": 20,

  "enable_time": true,
  "time_x": 20, "time_y": 40, "time_font_size": 32,

  "enable_cpu_bar": true,
  "cpu_bar_x": 20, "cpu_bar_y": 80,
  "cpu_bar_width": 440, "cpu_bar_height": 30,

  "enable_ram_bar": true,
  "ram_bar_x": 20, "ram_bar_y": 120,
  "ram_bar_width": 440, "ram_bar_height": 30,

  "enable_network": true,
  "network_x": 20, "network_y": 160,
  "network_font_size": 28,

  "enable_dynamic_text": true,
  "dynamic_text_font_color": [0, 255, 255],
  "format": "CPU: {cpu_percent}% | RAM: {ram_percent}% | ↑{net_up} ↓{net_down} | {time}"
}

Dynamic Text Placeholders

{cpu_percent}CPU usage % {ram_percent}RAM usage % {net_up}Upload speed (KB/s) {net_down}Download speed (KB/s) {loadavg}Load average (1/5/15 min) {time}Current time {date}Current date

Custom Fonts

Create a fonts/ folder and place .ttf files there. Recommended free fonts:

Digital-7— Classic 7-segment display look
DS-Digital— Clean digital readout style
Orbitron— Futuristic geometric sans

Running as Daemon

Create /etc/systemd/system/ttlcd.service:

[Unit]
Description=Thermaltake LCD Dashboard
After=network.target

[Service]
Type=simple
User=root
ExecStart=/path/to/ttlcd /path/to/config.json
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
sudo systemctl enable ttlcd.service
sudo systemctl start ttlcd.service

Troubleshooting

BLACK SCREEN Check console for "Device initialization sequence completed"
USB TIMEOUT Detailed libusb error messages printed to console
NO FONT Falls back to Hershey — check font path and FreeType support
PERMISSION Run with sudo for USB device access
