// Thermaltake Tower 200 · USB LCD driver · Linux · C++20

langC++20 platformLinux licenseMIT buildpassing packagesdeb · rpm · pkg version1.0.0
Modern C++ port of the original Python driver for the Thermaltake Tower 200 (and compatible models) 3.9-inch USB LCD panel at 480×128 resolution. Provides a highly customizable, performant system monitoring dashboard with live updates, custom TTF fonts, bar graphs, network speed, and more — running natively on Linux with full USB protocol support.
Resolution480×128
Panel Size3.9″
StandardC++20
InterfaceUSB HID
ConfigJSON

✦ Features

Live CPU & RAM usage bars with configurable dimensions
Real-time network upload/download speed display
Custom TTF fonts via OpenCV FreeType (Digital-7, Orbitron…)
Dynamic text widget with powerful format placeholders
Full JSON configuration — layout, colors, widgets
Complete USB init sequence & robust libusb error reporting
Graceful fallback — shows ERR if a data source fails
Temporary image rendering with automatic cleanup

◈ Requirements

Runtime

sudo apt install libusb-1.0-0 libopencv-core4.5 libopencv-imgproc4.5 \
    libopencv-imgcodecs4.5 libopencv-highgui4.5

Build

sudo apt install build-essential pkg-config libusb-1.0-0-dev \
    libopencv-dev libfreetype6-dev

⬡ Building

make -j$(nproc)           # build the binary
sudo ./ttlcd config.json  # run (USB requires sudo)

The makefile auto-generates Version.hpp, CMakeLists.txt, all packaging files, and the GitHub Actions workflow on every build.

Clean targets

make clean            # remove build artifacts
make clean-packaging  # remove generated packaging files
make clean-all        # both

↓ Install

wget https://github.com/YOURUSERNAME/ttlcd/releases/latest/download/ttlcd-v1.0.0-amd64.deb
sudo dpkg -i ttlcd-v1.0.0-amd64.deb
sudo rpm -i ttlcd-1.0.0-1.x86_64.rpm
# Download PKGBUILD.tar.gz from Releases, then:
tar xzf ttlcd-v1.0.0-PKGBUILD.tar.gz
makepkg -si

≡ Configuration

All layout and widget settings are controlled via config.json passed at runtime.

Full config.json example 
{
  "idVendor":   9802,
  "idProduct":  9021,
  "use":        "NODE",
  "orientation":"TOP",
  "background": "./bg.jpg",

  "font_size":  14,
  "font_color": [255, 255, 255],
  "font_file":  "fonts/Digital-7.ttf",

  "enable_date":           true,
  "date_x": 20,            "date_y": 10,   "date_font_size": 20,

  "enable_time":           true,
  "time_x": 20,            "time_y": 40,   "time_font_size": 32,

  "enable_cpu_bar":        true,
  "cpu_bar_x": 20,         "cpu_bar_y": 80,
  "cpu_bar_width": 440,    "cpu_bar_height": 30,

  "enable_ram_bar":        true,
  "ram_bar_x": 20,         "ram_bar_y": 120,
  "ram_bar_width": 440,    "ram_bar_height": 30,

  "enable_network":        true,
  "network_x": 20,         "network_y": 160,  "network_font_size": 28,

  "enable_loadavg":        true,
  "loadavg_x": 20,         "loadavg_y": 200,

  "enable_dynamic_text":   true,
  "dynamic_text_x": 20,    "dynamic_text_y": 230,
  "dynamic_text_font_size": 24,
  "dynamic_text_font_color": [0, 255, 255],
  "format": "CPU: {cpu_percent}% | RAM: {ram_percent}% | ↑{net_up} ↓{net_down} | {time}"
}

Key reference

KeyTypeDescription
idVendor / idProductintUSB vendor & product IDs for your LCD panel
orientationstring"TOP" or "BOTTOM" — physical mount orientation
backgroundstringPath to background image (jpg/png)
font_filestringPath to custom TTF; falls back to Hershey if missing
font_color[B,G,R]Default text color — note OpenCV uses BGR order
enable_*boolToggle any widget on or off
*_x / *_yintWidget position in pixels from top-left
formatstringDynamic text format string using placeholders below

Dynamic text placeholders

{cpu_percent}CPU usage percentage {ram_percent}RAM usage percentage {net_up}Upload speed in KB/s {net_down}Download speed in KB/s {loadavg}Load average — 1 / 5 / 15 min {time}Current time {date}Current date

⚙ Running as a Daemon

Create /etc/systemd/system/ttlcd.service:

[Unit]
Description=Thermaltake LCD Dashboard
After=network.target

[Service]
Type=simple
User=root
ExecStart=/usr/bin/ttlcd /etc/ttlcd/config.json
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
sudo systemctl enable --now ttlcd.service

! Troubleshooting

Black screen Check console for "Device initialization sequence completed"
USB timeout Detailed libusb error codes are printed to stderr
No font Falls back to Hershey — verify font path and FreeType support in your OpenCV build
Permission denied Run with sudo, or add a udev rule: SUBSYSTEM=="usb", ATTR{idVendor}=="2653", MODE="0666"
Wrong colors OpenCV uses BGR order — color arrays are [B, G, R] not RGB
